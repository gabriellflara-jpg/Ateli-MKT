# Ateliê MKT

Sistema de 37 skills de IA para agências e freelancers de marketing — execução e
estratégia, com memória automática por cliente. Roda no Claude Cowork, Claude Code e
Claude Chat.

## O que tem aqui

- `.claude/skills/` — as 37 skills, no padrão oficial do Claude Code (auto-descoberta,
  "Ative o [nome]" funciona direto):
  - 22 de execução: carrossel (PNG real), sites (HTML + banco de design ui-ux-pro-max),
    SEO, Google Ads, Meta Ads, Google Meu Negócio, WhatsApp, design, copy
  - 10 de estratégia: ofertas (Hormozi), StoryBrand, funil de lançamento, precificação,
    negociação, diagnóstico de negócio
  - 4 de sistema/memória: instalar, criar cliente, abrir cliente, atualizar
  - 1 banco de design de terceiros (ui-ux-pro-max, MIT, licença preservada dentro da skill)
- `clientes/_template/` — molde usado pra criar a pasta de cada cliente novo
- `_memoria/` — memória da própria agência (não confundir com memória de cliente)
- `COMO-INSTALAR.md` — guia de instalação completo

## Instalação rápida

Ver `COMO-INSTALAR.md` para o passo a passo completo (Cowork, Claude Code ou Chat).
