# Ateliê MKT

Sistema de 36 agentes de IA para agências e freelancers de marketing — execução e
estratégia, com memória automática por cliente. Roda no Claude Cowork, Claude Code e
Claude Chat.

## O que tem aqui

- `execucao/` — 22 agentes de entrega: carrossel (PNG real), sites, SEO, Google Ads,
  Meta Ads, Google Meu Negócio, WhatsApp, design, copy
- `estrategia/` — 10 agentes de estratégia: ofertas (Hormozi), StoryBrand, funil de
  lançamento, precificação, negociação, diagnóstico de negócio
- `_sistema/` — 4 agentes de memória: instalar, criar cliente, abrir cliente, atualizar
- `clientes/_template/` — molde usado pra criar a pasta de cada cliente novo
- `_memoria/` — memória da própria agência (não confundir com memória de cliente)
- `COMO-INSTALAR.md` — guia de instalação

## Instalação rápida

Ver `COMO-INSTALAR.md` para o passo a passo completo (Cowork, Claude Code ou Chat).
